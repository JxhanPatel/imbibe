# 1340. Jump Game V 🔴

**Topics:** Dynamic Programming, DFS, Memoization, Array




## Intuition

Imagine you are standing on buildings of different heights.

You can only jump from a taller building to a shorter one. Also, while jumping, you cannot "see through" a taller or equal building in between because it blocks your path.

So from every index, we want to know:

> "What is the maximum number of places I can visit if I start from here?"

This becomes a classic DFS + memoization problem:

* Try all valid jumps from an index
* Recursively compute the best path
* Save the answer so we do not recompute it again

It is similar to finding the longest path in a graph.






# Approach

We use DFS with memoization.

For every index `i`:

1. Explore left up to distance `d`
2. Stop if:

   * index goes out of bounds
   * we find an element greater than or equal to `arr[i]`
3. Do the same on the right side
4. Take the best possible jump

The recurrence is:

$$
dp[i] = 1 + \max(dp[j])
$$

where `j` is a valid index reachable from `i`.

The `1` counts the current index itself.

Memoization helps avoid recalculating states repeatedly.






# Example Walkthrough

```python
arr = [6,4,14,6,8,13,9,7,10,6,12]
d = 2
```

Suppose we start at index `10` value `12`.

Possible jumps:

* index `8` value `10`
* cannot jump to index `9` then beyond because rules apply carefully

From `8`:

* jump to `6`

From `6`:

* jump to `7`

Path becomes:

```text
10 -> 8 -> 6 -> 7
```

Total visited indices:

```text
4
```





# Code

```python
from typing import List
from functools import lru_cache

class Solution:
    def maxJumps(self, arr: List[int], d: int) -> int:

        n=len(arr)

        @lru_cache(None)
        def dp(i):
            res=1

            j = i-1
            while j>=0 and j>=i-d:
                if arr[j] >= arr[i]:
                    break

                res = max(res, 1+dp(j))
                j-= 1


            for j in range(i+1, min(i+d+1, n)):
                if arr[j] >= arr[i]:
                    break
                res = max(res, 1+dp(j))

            return res


        return max(dp(i) for i in range(n))
```





# Complexity

* Each index explores at most `d` positions left and right
* Every state is computed once

Time Complexity:

$$
O(n \cdot d)
$$

Space Complexity:

$$
O(n)
$$

for recursion stack and memoization.



# Edge Cases

1. All elements equal

```python
arr = [3,3,3,3]
```

No jumps possible.

Answer:

```text
1
```



2. Strictly decreasing array

```python
arr = [7,6,5,4]
```

You can keep jumping forward.



3. `d = 1`

Only adjacent jumps allowed.



# Lessons

* DFS + memoization is powerful for "best path from index" problems
* Breaking when encountering a taller/equal element is important
* Think of the array as a directed graph
* Memoization converts exponential recursion into efficient DP
* Longest path style problems often use:

  * recursion
  * caching
  * transition exploration
