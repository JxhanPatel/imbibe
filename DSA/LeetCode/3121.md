# 3121. Count the Number of Special Characters II 🟡

**Topics:** String, Hash Table





## Intuition

Think of each letter as a student entering a classroom.

* Lowercase letters (`a`) are students entering through the front door.
* Uppercase letters (`A`) are the same students entering later through the back door.

A letter is **special** only if:

1. The student entered through the front door at least once.
2. The same student later entered through the back door.
3. No front-door entry happens after the first back-door entry.

So for every character, we want:

* all lowercase appearances before the first uppercase appearance.

This becomes much easier if we track:

* the **last lowercase position**
* the **first uppercase position**

for every letter.








## Approach

We will use two arrays:

* `last_lower[26]`

  * stores the last index where lowercase appeared
* `first_upper[26]`

  * stores the first index where uppercase appeared

### Steps

1. Traverse the string once.
2. For lowercase letters:

   * keep updating their last position.
3. For uppercase letters:

   * only store the first occurrence.
4. Finally, for each character:

   * it is special if:

     * lowercase exists
     * uppercase exists
     * `last lowercase index < first uppercase index`








## Example Walkthrough

### Input

```python
word = "aaAbcBC"
```

### Traverse

| Index | Char | Action             |
| ----- | ---- | ------------------ |
| 0     | a    | last_lower[a] = 0  |
| 1     | a    | last_lower[a] = 1  |
| 2     | A    | first_upper[a] = 2 |
| 3     | b    | last_lower[b] = 3  |
| 4     | c    | last_lower[c] = 4  |
| 5     | B    | first_upper[b] = 5 |
| 6     | C    | first_upper[c] = 6 |

Now check:

* `a → 1 < 2` ✅
* `b → 3 < 5` ✅
* `c → 4 < 6` ✅

Answer = `3`









## Code

```python
class Solution:
    def numberOfSpecialChars(self, word: str) -> int:

        last_lower = {}
        first_upper = {}

        for i, ch in enumerate(word):
            if ch.islower():
                last_lower[ch] = i


            elif ch not in first_upper:
                first_upper[ch] = i

        special=0

        for ch, pos in last_lower.items():
            upper = ch.upper()

            
            if upper in first_upper and pos < first_upper[upper]:
                special+= 1

        return special
```








## Complexity

* **Time:** $O(n)$
* **Space:** $O(1)$

Only fixed-size arrays of length 26 are used.








## Edge Cases

1. **Only lowercase letters**

```python
"abc"
```

No uppercase exists → answer is `0`

2. **Uppercase appears first**

```python
"AbBCab"
```

Lowercase comes after uppercase → not special

3. **Repeated lowercase before uppercase**

```python
"aaA"
```

Still valid because all lowercase letters appear before uppercase

4. **Mixed invalid order**

```python
"aAa"
```

Last lowercase comes after uppercase → invalid








## Lessons

* Tracking positions is often easier than simulating conditions repeatedly.

* "All lowercase before uppercase" can be converted into:

  `last lowercase index < first uppercase index`

* Fixed alphabet size (`26`) allows constant-space solutions.

* This is a great example of reducing a string problem into index comparisons.
