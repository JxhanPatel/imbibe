# 2657. Find the Prefix Common Array of Two Arrays 🟡

**Topics**: Array, Hash Table, Counting

## Intuition

Imagine two people collecting cards one by one.

* `A[i]` is the card picked by person A
* `B[i]` is the card picked by person B

After every step, we want to know:

> "How many card numbers have been seen by BOTH people so far?"

So we keep track of which numbers have appeared already.

Whenever a number becomes visible in both arrays, we increase our common count.



## Approach

We use two sets:

* `seenA` → numbers seen in array A
* `seenB` → numbers seen in array B

At each index `i`:

1. Add `A[i]` to `seenA`
2. Add `B[i]` to `seenB`
3. Check:

   * if `A[i]` already existed in `seenB`, it is now common
   * if `B[i]` already existed in `seenA`, it is now common
4. Be careful:

   * if `A[i] == B[i]`, count it only once

We maintain a running `common` count and store it in the answer array.


## Example Walkthrough

### Input

```python
A = [1,3,2,4]
B = [3,1,2,4]
```

### Step-by-step

| i | A[i] | B[i] | Common Numbers | Count |
| - | ---- | ---- | -------------- | ----- |
| 0 | 1    | 3    | none           | 0     |
| 1 | 3    | 1    | 1, 3           | 2     |
| 2 | 2    | 2    | 1, 2, 3        | 3     |
| 3 | 4    | 4    | 1, 2, 3, 4     | 4     |

Result:

```python
[0,2,3,4]
```



## Code

```python
class Solution:
    def findThePrefixCommonArray(self, A: List[int], B: List[int]) -> List[int]:
        seen = {}
        result = []

        common=0

        for a, b in zip(A, B):
            seen[a] = seen.get(a, 0)+1

            if seen[a] == 2:
                common += 1


            if a!=b:
                seen[b] = seen.get(b, 0)+1

                if seen[b]==2:
                    common+= 1

            else:
                common+= 1

            result.append(common)

        return result
```



## Complexity

* Time Complexity: $O(n)$
* Space Complexity: $O(n)$



## Edge Cases

1. Arrays of size 1

```python
A = [1]
B = [1]
```

2. No common elements until the end

```python
A = [1,2,3]
B = [3,1,2]
```

3. Same arrays

```python
A = [1,2,3]
B = [1,2,3]
```



## Lessons

* Sets are very useful for tracking previously seen elements.
* Running counts help avoid recalculating from scratch.
* Watch carefully for double counting when values are equal.
* Since arrays are permutations, every number eventually becomes common.
