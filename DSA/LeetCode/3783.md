# 3783. Mirror Distance of an Integer 🟢

**Topics:** Math, Number Manipulation







## Intuition

Think of a number like a word written on paper. If you flip the paper horizontally, the digits appear reversed.

Now imagine comparing the original number and this reversed version. The mirror distance is simply how far apart these two numbers are on the number line.

So the task becomes:

1. Reverse the digits
2. Find the absolute difference






## Approach

We will follow a simple process:

1. Reverse the number

   * Extract digits one by one using modulo `% 10`
   * Build the reversed number step by step

2. Compute the absolute difference

   * Use `abs(n - reversed_n)`

This avoids converting to string and keeps it purely mathematical.







## Example Walkthrough

### Example: `n = 25`

1. Reverse the number:

   * Take last digit: 5 → reversed = 5
   * Next digit: 2 → reversed = 52

2. Compute distance:

   * `abs(25 - 52) = 27`

Final answer: **27**







## Code

```python
class Solution:
    def mirrorDistance(self, n: int) -> int:

        original = n
        flipped = 0

        while n > 0:
            flipped = flipped * 10 + (n % 10)
            n //= 10


        return abs(original - flipped)
```







## Complexity

* Time: $O(d)$ where $d$ is number of digits
* Space: $O(1)$






## Edge Cases

1. Single digit number

   * Example: `7 → reverse = 7 → answer = 0`

2. Numbers ending with zero

   * Example: `10 → reverse = 1`

3. Maximum constraint

   * Works fine for up to $10^9$







## Lessons

* Reversing a number using math is a common pattern
* Always preserve the original value if you modify input
* Absolute difference problems are usually straightforward after transformation
