# 3689. Maximum Total Subarray Value I 🟡

**Topics:** Array, Greedy

## Intuition

The problem looks like we need to choose **exactly `k` subarrays** to maximize the total value.

A key observation changes everything:

* We are allowed to choose the **same subarray multiple times**.
* The value of a subarray is:

$$
\max(subarray) - \min(subarray)
$$

Since repetitions are allowed, if we find the subarray with the **largest possible value**, we can simply choose it `k` times.

So the problem becomes:

> Find the maximum value among all subarrays, then multiply it by `k`.

Now, what is the maximum possible value of any subarray?

For any subarray:

$$
\max(subarray) \le \max(nums)
$$

and

$$
\min(subarray) \ge \min(nums)
$$

Therefore:

$$
\max(subarray)-\min(subarray)
\le
\max(nums)-\min(nums)
$$

The entire array contains both the global maximum and global minimum, so the whole array achieves exactly:

$$
\max(nums)-\min(nums)
$$

Thus the maximum subarray value is simply:

$$
\max(nums)-\min(nums)
$$

and the answer is:

$$
k \times (\max(nums)-\min(nums))
$$

## Approach

1. Find the maximum element in `nums`.
2. Find the minimum element in `nums`.
3. Compute their difference.
4. Multiply the difference by `k`.
5. Return the result.

This works because the whole array is a valid subarray and already achieves the largest possible value.

## Example Walkthrough

### Example 1

```text
nums = [1,3,2]
k = 2
```

Maximum element:

```text
3
```

Minimum element:

```text
1
```

Maximum subarray value:

```text
3 - 1 = 2
```

Choose the whole array twice:

```text
2 + 2 = 4
```

Answer:

```text
4
```

### Example 2

```text
nums = [4,2,5,1]
k = 3
```

Maximum element:

```text
5
```

Minimum element:

```text
1
```

Maximum subarray value:

```text
5 - 1 = 4
```

Choose a maximum-value subarray three times:

```text
4 + 4 + 4 = 12
```

Answer:

```text
12
```

## Code

```python
class Solution:
    def maxTotalValue(self, nums: List[int], k: int) -> int:

        spread = max(nums)-min(nums)
        ans = spread*k
        return ans
```

## Complexity

Time complexity:

$$
O(n)
$$

Space complexity:

$$
O(1)
$$

## Edge Cases

1. **Single element array**

```text
nums = [5]
k = 100
```

Every subarray has value `0`.

Answer:

```text
0
```

2. **All elements equal**

```text
nums = [7,7,7,7]
k = 10
```

Maximum and minimum are equal.

Answer:

```text
0
```

3. **Very large k**

```text
nums = [1,1000000000]
k = 100000
```

Still works because we only compute:

```text
(max(nums) - min(nums)) * k
```

## Lessons

* Always pay attention to whether repeated selections are allowed.
* If an item can be chosen multiple times, many optimization problems reduce to finding the single best choice.
* The maximum possible value of `max - min` over all subarrays is achieved by the entire array.
* Sometimes a seemingly difficult subarray problem collapses into a simple global observation.
