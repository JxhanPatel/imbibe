# 2615. Sum of Distances 🟡

**Topics:** Arrays, Hash Map, Prefix Sum






## Intuition

Imagine students sitting in a row, and each student wants to know how far they are from every other student who shares the same favorite number.

If we do this naively, for each student, we scan the whole row to find matching values and compute distances. That would be slow.

Instead, think of grouping students by their favorite number. Now within each group, we only care about positions of that number.

If we already know the positions of all equal elements, we can cleverly compute distances using prefix sums instead of recomputing everything every time.







## Approach

1. Group indices by value using a hashmap
   Example:
   nums = [1,3,1,1,2]
   → {1: [0,2,3], 3: [1], 2: [4]}

2. For each group:

   * Use prefix sums to efficiently compute distance contributions.
   * For a position `i` in the group:

     * Left side contribution: distance to all previous indices
     * Right side contribution: distance to all future indices

3. Formula idea:

   * Let `pos` be sorted indices
   * Let `prefix[i]` = sum of first i indices

   For index `pos[i]`:

   * Left = `pos[i] * i - prefix[i]`
   * Right = `(prefix[n] - prefix[i+1]) - pos[i] * (n - i - 1)`

4. Add both contributions to get result for that index.








## Example Walkthrough

Input: nums = [1,3,1,1,2]

Group 1 → indices = [0,2,3]

Compute prefix sum:
prefix = [0, 0, 2, 5]

Now compute for each:

* i = 0 (pos = 0):
  Left = 0
  Right = (2+3) - 0*2 = 5
  → 5

* i = 1 (pos = 2):
  Left = 2*1 - 0 = 2
  Right = (3) - 2*1 = 1
  → 3

* i = 2 (pos = 3):
  Left = 3*2 - (0+2) = 6 - 2 = 4
  Right = 0
  → 4

Final result: [5,0,3,4,0]









## Code

```python
from collections import defaultdict


class Solution:
    def distance(self, nums: List[int]) -> List[int]:
    
        result = [0] * len(nums)

        groups = defaultdict(list)


        for i, num in enumerate(nums):
            groups[num].append(i)


        for group in groups.values():
            if len(group) == 1:
                continue


            prefix = [0]

            for idx in group:
                prefix.append(prefix[-1] + idx)



            for i in range(len(group)):
                idx = group[i]
                left = idx * i - prefix[i]
                right = (prefix[-1] - prefix[i+1]) - idx * (len(group) - i - 1)
                result[idx] = left + right


        return result
```







## Complexity

Time complexity:
[
O(n)
]

Space complexity:
[
O(n)
]






## Edge Cases

* All elements are unique → result is all zeros
* All elements are the same → maximum computations inside one group
* Single element array → output is [0]







## Lessons

* Grouping by value simplifies many problems
* Prefix sums help avoid repeated calculations
* Splitting contribution into left and right parts is a powerful trick
* Index-based math is often cleaner than brute force comparisons
