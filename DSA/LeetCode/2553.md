# 2553. Separate the Digits in an Array 🟢

**Topics:** Array







## Intuition

Imagine you have a box filled with numbers, and your task is to break every number into its individual digits while keeping everything in the same order.

For example:

* `13` becomes `[1, 3]`
* `25` becomes `[2, 5]`

You simply go number by number and collect all the digits one after another.

A simple way to do this is by converting each number into a string, then looping through its characters and turning them back into integers.








## Approach

We will use the following steps:

1. Create an empty result array called `answer`.
2. Traverse every number in `nums`.
3. Convert the number into a string.
4. Loop through each character in the string.
5. Convert each character back into an integer and append it to `answer`.
6. Return `answer`.

This works because strings let us access digits in the exact order they appear.











## Example Walkthrough

### Input

```python
nums = [13, 25, 83]
```

### Step-by-step

* Start with `answer = []`

#### Number = 13

* `"13"` → `'1'`, `'3'`
* Add `1`, `3`

`answer = [1, 3]`

#### Number = 25

* `"25"` → `'2'`, `'5'`
* Add `2`, `5`

`answer = [1, 3, 2, 5]`

#### Number = 83

* `"83"` → `'8'`, `'3'`
* Add `8`, `3`

`answer = [1, 3, 2, 5, 8, 3]`

### Final Output

```python
[1, 3, 2, 5, 8, 3]
```










## Code

```python
class Solution:
    def separateDigits(self, nums: List[int]) -> List[int]:
        answer = []

        for num in nums:

            for digit in str(num):
                answer.append(int(digit))

        return answer

```















## Complexity

* Time Complexity: $O(n \cdot d)$
  where `d` is the number of digits in each number.

* Space Complexity: $O(n \cdot d)$
  because we store all separated digits in the result array.












## Edge Cases

1. Single digit numbers

```python
nums = [7, 1, 3]
```

Output:

```python
[7, 1, 3]
```

2. Largest allowed number

```python
nums = [100000]
```

Output:

```python
[1, 0, 0, 0, 0, 0]
```

3. Array with one element

```python
nums = [42]
```

Output:

```python
[4, 2]
```









## Lessons

* String conversion can simplify digit-related problems.
* Preserving order is important in array traversal problems.
* Building a result list step by step is a common array technique.
* Sometimes the simplest solution is also the cleanest and most readable.
