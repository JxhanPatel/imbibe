# 1344. Angle Between Hands of a Clock 🟡

**Topics:** Math

## Intuition

Think of a clock as a circular track of **360 degrees**.

* The **minute hand** moves quickly and completes one full rotation in 60 minutes.
* The **hour hand** moves slowly and completes one full rotation in 12 hours.

A very common mistake is to assume the hour hand stays fixed until the next hour.

For example, at `3:30`, the hour hand is not exactly at `3`. It has already moved halfway toward `4`.

So the problem becomes:

1. Find the exact position of the minute hand.
2. Find the exact position of the hour hand.
3. Compute the angle between them.
4. Return the smaller angle because two angles are always formed.



## Approach

First, convert both hands into angles measured from the `12 o'clock` position.

### Minute hand

The minute hand covers:

```text
360 degrees in 60 minutes
```

So each minute contributes:

```text
360 / 60 = 6 degrees
```

Hence,

```python
min_angle = minutes * 6
```



### Hour hand

The hour hand covers:

```text
360 degrees in 12 hours
```

So each hour contributes:

```text
360 / 12 = 30 degrees
```

But the hour hand also moves continuously as minutes pass.

Each minute contributes:

```text
30 / 60 = 0.5 degrees
```

Therefore,

```python
hr_angle = hour * 30 + minutes * 0.5
```

Before this, we convert `12` into `0` because both represent the same position on the clock:

```python
hour = hour % 12
```



Now compute the absolute difference:

```python
diff = abs(hr_angle - min_angle)
```

If this difference is greater than `180°`, then the other side of the circle gives a smaller angle.

So:

```python
if diff > 180:
    diff = 360 - diff
```

Finally, return `diff`.



## Example Walkthrough

**Input**

```text
hour = 12
minutes = 30
```

### Step 1: Convert hour

```text
hour = 12 % 12 = 0
```

### Step 2: Minute hand angle

```text
30 × 6 = 180°
```

### Step 3: Hour hand angle

```text
0 × 30 + 30 × 0.5
= 15°
```

### Step 4: Difference

```text
|15 - 180|
= 165°
```

Since `165 < 180`, this is already the smaller angle.

**Output**

```text
165
```



## Code

```python
class Solution:
    def angleClock(self, hour: int, minutes: int) -> float:

        hour = hour%12

        min_angle = minutes*6
        hr_angle = hour*30 + minutes*0.5


        diff = abs(hr_angle - min_angle)
        if diff > 180:
            diff = 360-diff

        return diff
```



## Complexity

**Time Complexity**

$$
O(1)
$$

Only a constant number of arithmetic operations are performed.



**Space Complexity**

$$
O(1)
$$

No extra space is used.



## Edge Cases

1. **Exactly 12 o'clock**

```text
hour = 12
minutes = 0
```

Both hands point to `12`.

Answer:

```text
0
```



2. **Hands opposite each other**

```text
hour = 6
minutes = 0
```

* Hour hand = `180°`
* Minute hand = `0°`

Difference:

```text
180°
```

Answer:

```text
180
```



3. **Hour hand between two numbers**

```text
hour = 3
minutes = 15
```

* Minute hand = `90°`
* Hour hand = `97.5°`

Difference:

```text
7.5°
```

Answer:

```text
7.5
```



## Lessons

* Circular motion problems are often solved by converting positions into angles.
* The hour hand moves continuously, not only when the hour changes.
* When measuring distance on a circle, there are always two possible paths.
* A useful trick is:

```python
min(diff, 360 - diff)
```

or equivalently,

```python
if diff > 180:
    diff = 360 - diff
```

Both ensure we return the smaller angle between the two hands.
