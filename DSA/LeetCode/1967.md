# 1967. Number of Strings That Appear as Substrings in Word 🟢

**Topics:** Array, String

## Intuition

Imagine you have a checklist of words, and you want to see which of them can be found inside a larger paragraph.

Instead of breaking the paragraph into every possible substring, we simply take each word from the checklist and ask:

> "Does this word appear anywhere in the paragraph?"

Python already provides an efficient way to answer this using the `in` operator. If the pattern is found, we increase our count.

## Approach

We iterate through every string in `patterns`.

For each pattern:

1. Check whether it exists as a substring of `word` using:
   ```python
   pattern in word
   ```
2. If it does, increment the answer.
3. Return the final count after checking every pattern.

Since each pattern is checked independently, duplicate patterns are counted separately, exactly as required.

## Example Walkthrough

**Input**

```text
patterns = ["a","abc","bc","d"]
word = "abc"
```

### Step 1

Check `"a"`:

```text
"a" in "abc" → True
```

Count = 1

### Step 2

Check `"abc"`:

```text
"abc" in "abc" → True
```

Count = 2

### Step 3

Check `"bc"`:

```text
"bc" in "abc" → True
```

Count = 3

### Step 4

Check `"d"`:

```text
"d" in "abc" → False
```

Count remains 3.

**Output**

```text
3
```

## Code

```python
class Solution:
    def numOfStrings(self, patterns: List[str], word: str) -> int:
        count=0

        for pattern in patterns:

            if pattern in word:
                count+= 1

        return count
```

## Complexity

Let:

- $n$ = number of patterns
- $m$ = length of `word`
- $k$ = average length of a pattern

**Time Complexity**

$$
O(n \times m \times k)
$$

In the worst case, each substring search scans the word.

**Space Complexity**

$$
O(1)
$$

Only a few variables are used.

## Edge Cases

1. **Duplicate patterns**

```text
patterns = ["a","a","a"]
word = "ab"
```

Each `"a"` is counted separately.

Answer:

```text
3
```

2. **No pattern exists**

```text
patterns = ["x","y","z"]
word = "abc"
```

Answer:

```text
0
```

3. **Pattern equals the entire word**

```text
patterns = ["leetcode"]
word = "leetcode"
```

The whole string is also a substring.

Answer:

```text
1
```

## Lessons

- Python's `in` operator provides a clean way to check for substrings.
- When constraints are small, a straightforward solution is often the best choice.
- Read the problem carefully: duplicate patterns should each be counted if they appear.
- Always leverage built-in string operations before implementing more complex algorithms like KMP or Rabin-Karp.
