# 3754. Concatenate Non-Zero Digits and Multiply by Sum 🟢

**Topics:** String, Math

## Intuition

Imagine writing down only the meaningful digits of a number while skipping every `0`.

For example, if the number is:

```text
10203004
```

you ignore all the zeros and keep:

```text
1234
```

Now there are only two things left to do:

1. Add all the remaining digits together.
2. Multiply this sum by the newly formed number.

Since both values depend only on the non-zero digits, we can collect those digits first and then compute the final answer.


## Approach

We first convert the integer into a string so that each digit can be processed individually.

1. Use a list comprehension to collect only the non-zero digits.
2. If the list is empty, return `0` because no number can be formed.
3. Compute the sum of the collected digits.
4. Join the digits together to form the new integer.
5. Return the product of the new integer and the digit sum.

This solution is short, clean, and easy to understand.


## Example Walkthrough

**Input**

```text
n = 10203004
```

### Step 1: Collect non-zero digits

```text
['1', '2', '3', '4']
```

### Step 2: Compute digit sum

```text
1 + 2 + 3 + 4 = 10
```

### Step 3: Form the new number

```text
1234
```

### Step 4: Multiply

```text
1234 × 10 = 12340
```

**Output**

```text
12340
```


## Code

```python
class Solution:
    def sumAndMultiply(self, n: int) -> int:

        nonzero = [ch for ch in str(n) if ch !="0"]

        if not nonzero:
            return 0

        total = sum(int(ch) for ch in nonzero)
        num = int("".join(nonzero))
        
        return num*total
```


## Complexity

**Time Complexity**

$$
O(d)
$$

where (d) is the number of digits in `n`.

**Space Complexity**

$$
O(d)
$$

to store the non-zero digits.


## Edge Cases

1. **All digits are zero**

```text
n = 0
```

No non-zero digits exist.

Output:

```text
0
```


2. **Exactly one non-zero digit**

```text
n = 1000
```

* New number = `1`
* Digit sum = `1`

Output:

```text
1
```


3. **No zeros in the number**

```text
n = 987
```

* New number = `987`
* Digit sum = `24`

Output:

```text
23688
```


## Lessons

* Converting a number to a string makes digit-wise operations much easier.
* List comprehensions provide a concise way to filter elements.
* Functions like `sum()` and `"".join()` can simplify many digit manipulation problems.
* Always consider the case where filtering removes every element, resulting in an empty list.
