# 3120. Count the Number of Special Characters I 🟢

**Topics:** String, Hash Set

## Intuition

Think of lowercase and uppercase letters as two groups attending a party.

A letter becomes *special* only if both versions show up.
For example:

* `'a'` alone is not enough
* `'A'` alone is not enough
* both `'a'` and `'A'` together make `'a'` special

So the main idea is simple:

* keep track of all lowercase letters
* keep track of all uppercase letters
* count how many letters appear in both groups



## Approach

We can use two sets:

* one set for lowercase characters
* one set for uppercase characters

### Steps

1. Traverse the string.
2. If the character is lowercase, add it to the lowercase set.
3. If the character is uppercase, convert it to lowercase and add it to the uppercase set.
4. Count how many characters exist in both sets.

Using sets makes lookup very fast.



## Example Walkthrough

### Input

```python
word = "aaAbcBC"
```

### Step-by-step

| Character | Lower Set | Upper Set |
| --------- | --------- | --------- |
| a         | {a}       | {}        |
| a         | {a}       | {}        |
| A         | {a}       | {a}       |
| b         | {a,b}     | {a}       |
| c         | {a,b,c}   | {a}       |
| B         | {a,b,c}   | {a,b}     |
| C         | {a,b,c}   | {a,b,c}   |

Now both sets contain:

```python
a, b, c
```

Answer:

```python
3
```









## Code

```python
class Solution:
    def numberOfSpecialChars(self, word: str) -> int:
        
        seen_lower = set(ch for ch in word if ch.islower())
        seen_upper = set(ch.lower() for ch in word if ch.isupper())


        return len(seen_lower & seen_upper)
```






## Complexity

* Time Complexity: $O(n)$
* Space Complexity: $O(1)$

The alphabet size is fixed to 26 letters, so space remains constant.






## Edge Cases

1. Only lowercase letters

```python
word = "abc"
```

Answer: `0`

2. Only uppercase letters

```python
word = "ABC"
```

Answer: `0`

3. Repeated characters

```python
word = "aaaaAAA"
```

Answer: `1`






## Lessons

* Sets are great for checking presence efficiently.
* Converting uppercase letters to lowercase simplifies comparison.
* Intersection of sets is a clean way to find common elements.
